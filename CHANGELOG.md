# 更新日志

项目的重要变更会记录在这里。

## [Unreleased] - 2026-03-06

### 改进
- 优化预览面板中版本卡片的可发现性，让关键操作更容易被用户注意到。
- 重构全局字体与排版体系，改善中英文混排下的界面观感。

### 修复
- 修复应用内中英文切换后，部分文本不能即时刷新的问题。

### 变更明细
- 预览面板
  - 版本卡片上的操作入口改为更明显、更易点击的呈现方式。
  - 强化当前选中状态，并在展开区域补充操作引导，降低理解成本。
- 国际化
  - 聊天区和系统消息改为基于当前语言实时翻译，不再停留在切换前的语言。
  - 设置中的语言切换控件以及顶部语言指示会随 locale 立即刷新。
- 字体与排版
  - 增加更适合 Windows 环境下中英文混排的全局字体栈与回退策略。
  - 细化标题、正文、输入框、状态标签和面板标题的字重、字号、行高与字距。
